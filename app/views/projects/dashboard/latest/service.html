National Polygon Service

National Polygon Services provides information about the extent of registered land and property in England and Wales.

Description

The National Polygon Service contains three licensed, chargeable datasets. Each dataset is only available as part of the National Polygon Service:
●   National Polygon dataset
●   Title Descriptor dataset
●   Title Number and UPRN Look Up dataset

Your use of the National Polygon Service is governed by licence terms and conditions. The annual licence permits you to resell the data to clients for their internal use, whether commercial or non-commercial. Members of Public Sector Mapping Agreement (PSMA) and/or One Scotland Mapping Agreement (OSMA) are permitted to use the data for their core business use, under the terms of PSMA/OSMA.

The licence does not cover the use of third party rights, as Land Registry use Ordnance Survey data in the preparation of the polygons. If you need this, please contact Ordnance Survey for the relevant licence conditions.

National Polygon dataset

The National Polygon dataset shows the indicative shape and position of each boundary of a registered title for land and/or property in England and Wales. Every title whether freehold or leasehold has at least one index polygon.

This dataset contains more than 24 million titles and includes 28 million polygons. Land Registry index polygons are mapped against Ordnance Survey’s large scale map (MasterMap).

Read the full specification for the National Polygon dataset

Data provided

The data includes:
●   shape
●   polygon identification
●   title number
●   date when polygon was created
●   update date (date when all or part of the title was last updated)
●   version of polygon identification number
●   record status (indicates additions, changes and deletions to the records)

Dataset

This dataset is a complete extract of the current state of the data and contains all of the changes that have occurred during the month. The data is provided as a ESRI shapefile.

The options include:
●   complete file: download the data at any time during the year
●   change only update file: provided on the second working day of each month
●   validation file: enables you to verify the information has been downloaded correctly

View an example or download a dummy file.

Title Descriptor dataset

The Title Descriptor dataset describes the legal interest(s) that have been recorded against freehold and leasehold estates (property and/or land) registered in England and Wales. When an estate is registered it is assigned a unique title number, enabling individual records to be interrogated.

Read the full specification for the Title Descriptor dataset

Data provided

The data includes:
●   title number (unique number that identifies the registration)
●   estate interest (describes the legal estate or legal interest that has been registered)
●   class of title (describes Registrar’s confidence in the ownership of the title at the point of registration)
●   registered status (indicates if the title is registered)
●   record status (indicates additions, changes and deletions to the records)

Dataset

This dataset is a complete extract of the current state of the data and contains all of the changes that have occurred during the month. The data is provided as a CSV file.

The options include:
●   complete file: download the data at any time during the year
●   change only update file: provided on the second working day of each month
●   validation file: enables you to verify the information has been downloaded correctly

View an example or download a dummy file.

Title Number and UPRN Look Up dataset

The Title Number and UPRN Look Up dataset contains Land Registry title numbers and associated UPRNs (Unique Property Reference Numbers) for freehold and leasehold property/land registered in England and Wales.

UPRNs are unique twelve digit numbers assigned to every addressable location in the UK. The information is created by GeoPlace, a joint venture between local authorities, who name and number every street and property, and Ordnance Survey. Ordnance Survey’s AddressBase links both property and land information to a map whenever possible.

Title numbers and UPRNs can only be linked where a relationship between the title number and AddressBase address has been identified.

UPRNs provide a comprehensive, complete and consistent identifier throughout a property’s life. Cross-referenced with title numbers they enable files to be matched and merged with datasets from different sources to create valuable insight.

Read the full specification for the Title Number and UPRN Lookup dataset

Data provided

The data includes:
●   title number (unique number that identifies the registration)
●   unique Property Reference Number (identifier of a spatial address)

Dataset

This dataset is a complete extract of the current state of the data. The data is provided as a CSV file.

The options include:
●   complete file: download the data at any time during the year, complete file will be refreshed monthly
●   validation file: enables you to verify the information has been downloaded correctly

View an example or download a dummy file.
